### k8s  haproxy修改超时时间

```
1、
vim haproxy-ingress-kubernetes.yaml

apiVersion: v1
data:
  timeout-client: 10m
  timeout-connect: 10m
  timeout-server: 10m
kind: ConfigMap
metadata:
  name: haproxy-ingress
  namespace: "default"
 
2、
kubectl create -f haproxy-ingress-.yaml
kubectl get cm -n default

3、
kubectl edit deploy -n default haproxy-ingress
在/args下一行添加
        - --configmap=default/haproxy-ingress

kubectl delete pod

kubectl delete rs
edit image
```
